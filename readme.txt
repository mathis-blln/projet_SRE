Ce projet, réalisé par Mathis Bouillon et Cheryl Kouadio, s'articule de la façon suivante : 

- dans le notebook main.ipynb, vous trouverez l'ensemble des codes ainsi que les commentaires qui répondent 
  aux différentes questions du TP donné;

- le dossier "Interface" contient les fichiers nécessaires au lancement du dash. Afin de lancer ce dernier, il suffit 
  de run le fichier "interface.py". Les fichiers "TP_SRE.py" et "VaR_models.py" contiennent les différentes fonctions
  importées depuis interface.py;

- le dossier Others contient les différents supports de cours ainsi que les énoncés des TP/projet.
